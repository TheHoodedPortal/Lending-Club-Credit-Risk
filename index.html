Predictions use this project's fitted Stage 1 logistic model (standardised features) for probability of default and the observed average loss-given-default by grade for severity; the survival curve uses the Cox proportional-hazards fit. Selecting a grade auto-sets a typical interest rate, but you can override it.
For demonstration only. The Stage 2 severity (LGD) model is fitted using “months on book” — how long a loan has been active, a post-origination variable not known at application time — which is one reason the dashboard uses observed average loss by grade rather than that model's coefficients. Trained on unsecured consumer loans (Lending Club, 2007–2018); not a substitute for a lender's own underwriting.
